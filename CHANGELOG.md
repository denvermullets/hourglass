# What's new

## 2026-04-28
- **Changelog page.** A new in-app changelog view shows what's shipped recently.
- **Cleaner pasting.** Pasting into the editor now strips outside formatting so it picks up the channel's styling.
- **Markdown polish.** Quoted code blocks render at normal text size, and the accent color stays in sync across messages.

## 2026-04-21
- **Themed favicon.** The browser tab icon now matches your chosen color theme.
- **Mobile typing fixes.** The Return key inserts a new line on mobile, and larger text sizes scale cleanly without breaking layouts.
- **Notification polish.** The unread dot and message timestamps got tightened up.

## 2026-04-14
- **Markdown all the way through.** The composer supports lists, tables, and horizontal rules, and you can paste Markdown straight in and have it render correctly.
- **Edits keep your formatting.** @mentions and newlines are preserved when you edit a message.
- **New-user welcome flow.** First-time users get a guided sign-up and a welcome message waiting in #general.
- **Read tracking.** Sending a message now marks the channel as read for you so unread counts stay accurate.

## 2026-04-09
- **Direct messages.** You can now DM other users, with a pop-up notice when a new conversation comes in.
- **Channel linking.** Type a channel reference and it turns into a clickable link.
- **Quote replies.** Pull a previous message into your reply as a quote, with safeguards against accidental double-sends.
- **Better video playback.** The in-message video player got an overhaul.
- **Connection status.** A small notice appears when you disconnect or reconnect from the server.

## 2026-04-03
- **Admin channel management.** Admins can create channels, archive them, and reorganize them across categories — and everyone sees the changes live without refreshing.
- **Mark all as read.** Clear every unread notification at once from the notifications drawer.
- **Channel-level unread indicators.** A dot on the channel and on the browser tab tells you when there's something new.
- **Smarter message grouping.** Consecutive messages from the same user within 60 seconds are grouped together.
- **Profile + timezone settings.** Change your username from your profile, and set your timezone so timestamps make sense.
- **Larger uploads + .md files.** The message character limit went up and Markdown files are now allowed as attachments.

## 2026-04-02
- **More themes and text sizes.** Several new high-contrast color themes plus adjustable text sizing in settings.
- **Layout flip.** Move the sidebar to the other side of the screen if that suits you better.
- **Notifications.** Get notified about replies and threaded mentions, with deep links straight to the message.
- **Mobile composer polish.** Composer height, send button placement, edit actions, and avatar rendering all got cleaned up on phones.

## 2026-03-31
- **Initial release.** Sign up, create a server, set up channels, and start chatting.
- **Rich messages.** A Lexical-based editor with editing, threaded replies, file/image/video attachments, and drag-and-drop uploads.
- **Personalization.** Color themes, custom avatars, and clickable links in messages.
